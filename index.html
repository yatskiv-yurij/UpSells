Encourage repeat purchase

It works very simply:

For each purchase you make, you credit the buyer’s account with a certain number of Bonus Points

Convenience and benefit for everyone

The client can spend the accumulated Bonus points in the same way as real ones — they can pay for the % of the purchase you specified

Thus, each purchase partially pays for the next one, and if you implement it in your store, then the buyer is 90% likely to return to make a repeat purchase!

Features

Give points for a product, or groups created by taxonomies

You can fully manage the amount of bonuses accrued for the purchase of both individual products and groups.
Groups are products united by common taxonomies (Categories, tags, attributes, price, brands) that you can create and customize in an unlimited number.

Easy-to-manage control of user points balance

The plugin allows you to conveniently track, accrue and control the balance of bonus points both for each individual user and in bulk according to a certain user attribute. At the same time, for each accrual / write-off, you can specify a reason, send a newsletter, and much more.

Auto actions for accruing bonuses

In this section, you can create and configure automatic points accrual companies based on a trigger or conditions, followed by user notification via email or popup.
Set up auto bonuses for registration, birthday, users of a certain age, amount of purchases, time since registration, users who abandoned the cart, users who left a review and many other triggers and conditions.

Moreover, from the specified settings, a page of rules for using the bonus system is automatically generated, which are available to the user in his personal bonus account.
